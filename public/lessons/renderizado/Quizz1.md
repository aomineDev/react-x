# QUIZZ 1 – Verdadero / Falso: Renderizado condicional en React

1. El operador lógico && en React se usa para renderizar algo solo cuando la condición es verdadera.
   V / F

2. Un “early return” sirve para mostrar una UI alternativa cuando una condición NO se cumple, evitando renderizar el resto del componente.
   V / F

3. switch es la estructura más recomendada para manejar componentes que dependen de 2 o más condiciones simultáneas.
   V / F

4. En React, un valor false retornado en JSX se mostrará en pantalla como texto.
   V / F

5. Combinar && con early returns es válido y común para simplificar componentes con muchas verificaciones.
   V / F
