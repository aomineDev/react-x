

# React

## Historia

React fue creada por Jordan Walke, un ingeniero de Facebook, en el año 2013, con el fin de hacer las interfaces web más rápidas y fáciles de mantener. Desde entonces, se ha convertido en una de las librerías más populares del mundo, utilizada por millones de desarrolladores y empresas como Facebook, Instagram, Netflix, Airbnb y WhatsApp.

## ¿Qué es React?

React es una librería de JavaScript basada en componentes, que te permite construir interfaces de usuario de manera simple y organizada.

Imagina que estás construyendo con bloques de LEGO. Cada pieza es un componente: un botón, un formulario, una tarjeta, un menú. Cada componente tiene:

- **Su propio diseño** (cómo se ve)
- **Su propia lógica** (qué hace)
- **Puede reutilizarse** en diferentes partes de tu aplicación

Al igual que con LEGO, conectas estos componentes para crear interfaces completas: desde una simple página de perfil hasta aplicaciones complejas como Facebook o Instagram.

## ¿Por qué usar React?

### ~1~ Reutilizable
Un botón que crees una vez puede usarse en toda tu aplicación. ¿Necesitas 100 botones? Solo defines uno y lo reutilizas.

### ~2~ Organizado
Cada componente vive en su propio archivo, lo que hace tu código más limpio y fácil de mantener. Es como tener tu habitación ordenada: sabes dónde está cada cosa.

### ~3~ Actualizaciones eficientes
React actualiza solo las partes de la página que cambian, no todo. Esto hace que tus aplicaciones sean súper rápidas.

### ~4~ Sintaxis amigable
JSX (la sintaxis de React) mezcla HTML con JavaScript de forma natural, haciendo que escribir código sea más intuitivo.

## ¿Quién usa React?

Empresas gigantes confían en React para sus productos:

- **Facebook** e **Instagram** (donde nació React)
- **Netflix** (su interfaz de usuario)
- **Airbnb** (búsqueda y reservas)
- **WhatsApp Web** (la versión de navegador)
- **Uber Eats** (pedidos de comida)

Si estas empresas confían en React para millones de usuarios, imagina lo que puedes construir tú.
