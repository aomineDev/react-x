# Ecosistema de React

El ecosistema de React es el conjunto de herramientas, librerías y frameworks que rodean a React y permiten crear aplicaciones completas, enrutamiento, manejo de estado, estilos, peticiones HTTP y compilación.

## Principales herramientas del ecosistema

- React Router: Para crear rutas y navegar entre páginas dentro de un SPA.
- Hoocks: Para manejar el estado de los componentes.
- Tainwilnd: Framework CSS que permite escribir estilos usando clases utilitarias para diseñar interfaces más rápidas.
- Estados globales (Context API, Zustand): Herramientas para compartir estados entre componentes sin tener que pasarlos manualmente como props.
- Fech: Métodos para consumir datos desde APIs externas o servicios remotos.
- Vite: Herramienta para la creacion y compilacion de proyectos en React.
- Next.js: Framework avanzando basado en React.
