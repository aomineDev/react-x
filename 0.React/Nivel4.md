# Creacion de un proyecto en React

Para iniciar un proyecto en React necesitamos una herramienta que genere la estructura base y configure el entorno de desarrollo. Actualmente, existen dos opciones principales (una moderna y una antigua).

## Herramientas

- Vite: La herramienta más rápida y moderna para crear proyectos con React.
  Permite compilar, optimizar y levanta el servidor de desarrollo en segundos.

```
npm create vite
```

- Cra: Herramienta oficial, para crear proyectos en react, pero ya no es muy usado debido a su lentitud

```
npx create-react-app nombre_de_tu_app
```

Nota !!!

```
Actualmente, todos los proyectos modelos utilizan vite por su rapides, Cra esta quedando obsoleto
```
